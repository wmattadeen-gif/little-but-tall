# Little But Tall Preview

A modern Next.js preview build for **Little But Tall Charities**, created as a direction for a full nonprofit website rebuild.

## What is included

- Multi-page Next.js site
- Homepage, About, Programs, C.L.I.M.B., AI & Life Skills, International Missions, Get Involved, and Contact pages
- Visual design direction with gradient feature panels, impact sections, and story cards
- Ready for Vercel deployment

## Local development

```bash
npm install
npm run dev
```

Open `http://localhost:3000`

## Production build

```bash
npm run build
npm run start
```

## Deploy to Vercel

### Option 1, easiest
1. Push this folder to a GitHub repo
2. Import the repo into Vercel
3. Vercel will detect Next.js automatically
4. Click deploy

### Option 2, Vercel CLI
```bash
npm install -g vercel
vercel
```
